Nmod

Calculates the modulus (remainder) of two decimal numbers using the formula: A - B * ⌊A / B⌋. Unlike the standard operation A mod B = A - B * (A \ B) in AL, which uses truncated integer division and may yield incorrect results when A is negative, this implementation uses the Floor function to ensure mathematically correct results even with negative values.

procedure Nmod(A: Decimal, B: Decimal): Decimal

Parameters

Name Type Description
A Decimal

The dividend (can be positive or negative).
B Decimal

The divisor (must be non-zero).

Returns

Type Description
Decimal

The remainder of the division of A by B.

Remarks

This formula guarantees that the sign of the result is consistent with the divisor, complying with the mathematical definition of the modulus.
