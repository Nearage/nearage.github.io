SetBlank

Sets up a blank structure based on the specified number of rows and page size. Iterates from the result of NewIndex to the calculated modulus value using the Mathx.Nmod function, creating new blank records in each iteration.

procedure SetBlank(Rows: Integer, Size: Integer): Integer

Parameters

Name Type Description
Rows Integer

The number of rows in the dataset.
Size Integer

The page size in standard row units. It can be calculated using the formula ⌊(p - h - f) / l⌋, where:

  • p is the total page height.
  • h is the header height.
  • f is the footer height.
  • l is the line height.

Returns

Type Description
Integer

An integer representing the final index value after the operation.

Use

Initializes a blank structure based on the specified parameters.

procedure Use(Format: Decimal, Length: Decimal, Static: Decimal, Repeat: Decimal, StdLin: Decimal): Integer

Parameters

Name Type Description
Format Decimal

The total page height.
Length Decimal

The number of rows in the dataset.
Static Decimal

The height of the static content.
Repeat Decimal

The height of the repeated content.
StdLin Decimal

The standard line height, which is the height of a single line in the dataset.

Returns

Type Description
Integer

The final index value after the operation.

Remarks

All specified parameters must be in the same unit of measurement (using inches is recommended).
